# SwiftUIBottomSheet

シンプルなメディア選択用ボトムシートを提供するSwiftUIライブラリ

## 特徴

- 4種類のメディアタイプ選択（写真、カメラ、動画、音声）
- ダークモード対応
- iOS 14.0以上をサポート
- デリゲートパターンによる簡単な実装

## インストール方法

### Swift Package Manager

`Package.swift`に以下を追加:

```swift
dependencies: [
    .package(url: "https://github.com/yuji-nakamoto/SwiftUIBottomSheet.git", from: "1.0.0")
]
